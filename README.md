# mini_app_for_food

Консольное приложение которое модет выполнять две функции.
    1. пше 

#### Описание файлов
1. `recipes.ipynb`
    - Файл содержащий часть иследования.
    - Выбор наилучшей модель. 
    - Парсинг API USDA
2. `recipes.py`
    - Файл содержащий основные классы проекта.
    - Main_class - класс отвечает за подбро бляда по набору продуктов. 
    - Dayly_menu - класс отвечает за подбор дневного меню. 
3. `nutritionist.py`
    - Файл содержащий основную программу проекта.
    - Используются классы написанные в `recipes.py`.
4. `model.joblib`
    - Файл содержит модель выбранную в `recipes.ipynb`. 
5. `requirements.txt`
    - Файл зависимостей
